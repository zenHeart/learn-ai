# RAG

解决模型挂载外部知识库的问题


## 核心流程
1. 准备阶段
  1. 数据源 `.md、.csv` 等资源
  2. 分段分片
  3. 向量化构建索引
  4. 向量索引库
2. 检索阶段
  1. 用户输入问题
  2. 检索向量库
  3. 返回相关文档
3. 生成阶段
  1. 将检索到的文档和用户问题拼接成 prompt，例如
      ```
      1. 你是一个知识库助手
      2. 请根据以下文档回答问题
      3. 文档：{{doc}}
      4. 问题：{{question}}
      ```
  2. 调用模型生成答案
  3. 返回答案
