# Context 注入方案

## 核心要点

1. **Context 是 AI 应用的天花板**：模型能力决定下限，Context 质量决定上限；好的 Context 能让普通模型超越未优化的高配模型
2. **分层注入策略**：
   - System Prompt（全局规则）→ 固定不变的指令
   - 动态 Context（用户输入、历史对话、领域知识）→ 每次请求更新
   - 工具输出（Tool Result）→ 按需注入，避免塞满 Context
3. **注入时机**：早注入高价值信息（用户意图、关键约束），延迟低价值信息（补充说明、参考文档）
4. **Context 压缩**：长对话需压缩历史，保留核心意图和关键信息，避免稀释
5. **质量 > 数量**：精心挑选的少而精信息，远优于大量噪音数据
