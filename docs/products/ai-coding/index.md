## The 5 Levels of AI Coding

| Level | Capability | Example Popular Products |
|-------|------------|-------------------------|
| L1   | Code-level completion   | GitHub Copilot, Tabby |
| L2   | Task-level code generation | Ticket to Code, IDE with Chat: ChatGPT, Claude, aider, cline, 16x Prompt, Cursor, Continue, PearAI, Windsurf |
| L3   | Project-level generation   | Ticket to PR, Prompt to UI: Codegen, Sweep, Pythagora, Plandex, v0 |
| L4   | PRD to production   | AI Software Engineers: Marblism, bolt.new, Trickle, Lovable, Devin, Genie, Engine, devlo, Gru |
| L5   | AI development team  | AutoDev, MetaGPT, MGX |

## Choosing Your AI Coding Tool

### For Beginners
**Recommended**: [Cursor](./cursor.md)
- Easiest to learn
- All-in-one solution (autocomplete + chat)
- Great multi-file context

### For VS Code Power Users
**Recommended**: [GitHub Copilot](./copilot.md)
- Seamless VS Code integration
- Fast autocomplete
- Industry standard

### For Terminal Lovers
**Recommended**: [Claude CLI](./claude-cli.md)
- Command-line first
- Excellent reasoning
- Great for refactoring

### For Google Ecosystem
**Recommended**: [Gemini CLI](./gemini-cli.md)
- Large context window
- Multimodal capabilities
- Google integration
