# 技能 (Skills)

技能是可移植、可重用的模块，将专业知识、工作流和能力打包，任何智能体都可以按需加载和使用。

## 技能列表

| 技能 | 描述 | 适用场景 |
|------|------|---------|
| [OpenSpec](./openspec.md) | 规格驱动开发框架 | AI 编程、需求管理、代码重构 |

## 技能核心概念

### 渐进式披露

技能分三个阶段加载，保持智能体快速：

1. **发现**（启动时）：仅加载名称 + 描述（50-100 tokens）
2. **激活**（相关时）：加载完整指令（<5000 tokens）
3. **执行**（按需）：根据需要访问捆绑的脚本和资源

### 技能格式

```markdown
# 技能名称

## 描述
技能的功能描述

## 使用方式
如何使用这个技能

## 示例
具体使用示例
```

## 创建自定义技能

1. 创建 SKILL.md 文件
2. 定义技能描述和使用方式
3. 添加示例和最佳实践
4. 放置在智能体可访问的路径
