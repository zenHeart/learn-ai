# 知乎 AI 落地讨论

> 来源：https://zhuanlan.zhihu.com/p/1909272469153477214

## 核心要点

1. **场景选择 > 技术选型**：AI 落地的第一步是找到高价值、低风险的业务场景，而非追逐最新模型
2. **数据是瓶颈**：大多数企业 AI 落地卡在数据质量而非模型能力，数据治理应先于模型调优
3. **人机协作模式**：AI 适合"草稿 + 人工审核"模式，完全自动化在 B 端场景风险过高
4. **成本可控**：按调用量付费的 API 模式降低了 AI 落地门槛，应优先验证 ROI 再投入固定成本
5. **渐进式推进**：从单点场景突破，形成 AI 能力后再逐步扩展，避免大而全的失败
