# 资源与社区

精心策划的工具和库列表，加速你的 AI 之旅。

## UI 组件库

更快地构建漂亮的聊天界面。

- **[Shadcn UI](https://ui.shadcn.com/)**: 复制/粘贴 React 组件的黄金标准。
- **[Vercel AI SDK UI](https://sdk.vercel.ai/docs/guides/ui)**: 官方无头 Hooks。
- **[NextUI](https://nextui.org/)**: 美观、快速的 React 组件。

## AI 库 (TypeScript)

- **[LangChain.js](https://js.langchain.com/)**: 编排。
- **[LlamaIndex.ts](https://ts.llamaindex.ai/)**: RAG 和数据。
- **[Transformers.js](https://huggingface.co/docs/transformers.js)**: 浏览器内机器学习。

## 向量数据库

- **[Supabase (pgvector)](https://supabase.com/vector)**: Postgres + 向量。
- **[Pinecone](https://www.pinecone.io/)**: 专用向量 DB (Serverless)。
- **[Upstash Vector](https://upstash.com/vector)**: 非常适合边缘部署。

## 学习

- **[DeepLearning.AI](https://www.deeplearning.ai/)**: Andrew Ng 的课程 (概念重)。
- **[Vercel AI SDK 文档](https://sdk.vercel.ai/docs)**: 最好的实战文档。
- **[Hugging Face 课程](https://huggingface.co/course)**: 最适合理解模型。
