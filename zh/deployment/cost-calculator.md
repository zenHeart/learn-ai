# AI 成本计算器

由于 Token 比例的原因，计算 AI 成本很复杂。

## 黄金公式

$$
\text{总成本} = (\text{用户数} \times \text{请求数/用户}) \times [(\text{输入 Token} \times P_{in}) + (\text{输出 Token} \times P_{out})]
$$

## 场景示例：RAG 聊天机器人

- **用户**: 1,000 日活跃用户。
- **请求**: 每个用户 10 条消息。
- **输入**: 2,000 Token (来自 PDF 的上下文) = $5.00 / 1M Token。
- **输出**: 500 Token (回答) = $15.00 / 1M Token。

**每日成本**:
1.  请求数 = 10,000
2.  输入成本 = 10k * (2000/1M * $5) = $100
3.  输出成本 = 10k * (500/1M * $15) = $75
4.  **总计 = $175 / 天** ($5,250 / 月)

## ROI 分析 (投资回报率)

这值得吗？

**客服机器人示例**:
- 成本: $5,250 / 月。
- 人工客服: $4,000 / 月。
- 如果机器人替代了 **2 名人工客服**，你每月节省 $2,750。
