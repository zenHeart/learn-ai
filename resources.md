# Resources & Community

A curated list of tools and libraries to accelerate your AI journey.

## UI Component Libraries

Build beautiful chat interfaces faster.

- **[Shadcn UI](https://ui.shadcn.com/)**: The gold standard for Copy/Paste React components.
- **[Vercel AI SDK UI](https://sdk.vercel.ai/docs/guides/ui)**: Official headless hooks.
- **[NextUI](https://nextui.org/)**: Beautiful, fast React components.

## AI Libraries (TypeScript)

- **[LangChain.js](https://js.langchain.com/)**: Orchestration.
- **[LlamaIndex.ts](https://ts.llamaindex.ai/)**: RAG and Data.
- **[Transformers.js](https://huggingface.co/docs/transformers.js)**: In-browser ML.

## Vector Databases

- **[Supabase (pgvector)](https://supabase.com/vector)**: Postgres + Vector.
- **[Pinecone](https://www.pinecone.io/)**: Specialized Vector DB (Serverless).
- **[Upstash Vector](https://upstash.com/vector)**: Great for edge deployments.

## Learning

- **[DeepLearning.AI](https://www.deeplearning.ai/)**: Andrew Ng's courses (Concept heavy).
- **[Vercel AI SDK Docs](https://sdk.vercel.ai/docs)**: Best practical docs.
- **[Hugging Face Course](https://huggingface.co/course)**: Best for understanding models.
