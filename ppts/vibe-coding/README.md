Vibe Coding 会话目的

本次 Vibe Coding（氛围式协作编码）会话旨在帮助团队理解并实践如何在日常开发中有效使用 AI 工具，提升工程效率，并建立稳定的 AI 辅助开发工作流。

核心目标：

- 让开发者理解什么是 Vibe Coding，掌握 AI 辅助编程的核心理念与协作方式。
- 熟悉主流 IDE 插件与终端工具在 AI 编程中的常见功能与最佳实践（如代码补全、重构建议、测试生成、解释与审查等）。
- 学会在真实开发场景中应用 AI 工具（修复 bug、重构、写测试、代码审查与文档编写等），并能将其自然整合到日常工作流中。 

本会话产出（可衡量项）：

- 一套可复用的 Vibe Coding 流程模板（包含角色、节奏与示例 prompts），便于团队复现和训练。
- 若干实操示例与代码片段，覆盖常见开发场景的 prompt 模板。
- 建议的工具清单与配置要点（IDE、终端、本地 LLM 及安全建议）。

后续建议：在会后进行 2 次短小的回顾会（15min），收集团队反馈并将常用 prompts 纳入内部知识库。
