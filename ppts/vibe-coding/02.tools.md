# IDE 插件类工具

## GitHub Copilot

- **代码补全**：Tab 键接受建议
- **内联模式**：⌘I 唤起内联窗口
- **对话模式**：⌃⌘I 聊天窗口
- **Agent 模式**：自动编辑并执行任务

## Cursor

- AI 优先的代码编辑器
- 出色的多文件上下文支持
- 一站式解决方案

::right::

<div style="padding-left: 20px;">

## 核心功能

- 代码补全与模板生成
- 代码解释与注释生成
- Bug 定位与修复建议
- 自动生成测试用例
- 重构建议与优化提示

## 推荐配置

- 开启行内建议 + 快捷键
- 设置隐私/数据使用策略
- 为常用任务准备 prompt 模板

# 终端/CLI 工具

## Claude CLI

```bash
# 交互式模式
claude

# 单次查询
claude -p 'explain this codebase'

# 继续对话
claude -c
```

**常用命令**：

- `/clear` - 清除对话
- `/memory` - 编辑 CLAUDE.md
- `/mcp` - 管理 MCP 服务器

::right::

<div style="padding-left: 20px;">

## Gemini CLI

```bash
# 单次提问
gemini -p 'your prompt here'

# 代码审查
cat file.py | gemini -p "review"

# 生成提交信息
git diff --cached | gemini -p \
  "Write commit message"
```

**常用命令**：

- `/chat save <tag>` - 保存聊天
- `/chat resume` - 恢复聊天
- `/stats` - 显示统计

</div>

# 风险与安全建议

<div style="display: flex; gap: 30px; margin-top: 20px;">

<div style="flex: 1;">

## ⚠️ 潜在风险

- AI 可能生成错误或不安全的代码
- 敏感信息可能被发送到外部服务
- 过度依赖可能降低代码审查能力
- 生成的代码可能存在版权问题

</div>

<div style="flex: 1;">

## 🛡️ 安全策略

- **验证代码**：不直接合并未经审核的代码
- **保护隐私**：避免在 prompt 中包含密钥/用户数据
- **限额审计**：为第三方服务配置使用限额
- **本地 LLM**：敏感场景使用 Ollama 等本地方案

</div>

</div>

<br/>

> 💡 **核心原则**：把 AI 作为 "建议者" 而非 "裁决者"，将其产出作为候选解并由开发者验证。
