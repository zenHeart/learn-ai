# 会前准备

- **时长建议**：整体 40-60 分钟（含 Demo 与问答）
- **技术/环境要求**：
  - 已安装并熟悉的编辑器（推荐 VS Code）
  - 基本的 Git 与 Node.js 使用能力
  - 至少可访问一个 AI 助手（云端 ChatGPT/Claude 或本地 LLM / IDE 插件）

**会前清单（主讲/组织者）**：

- 提前准备好示例仓库或代码片段（小而能覆盖常见场景）
- 为参与者列出推荐工具与安装指引
- 设定演示角色（如：Driver、Navigator、AI 助手）

**学习目标（本次会话结束时）**：

- 理解 Vibe Coding 的节奏与角色分工
- 掌握 5 个常见的 AI 编程应用场景及对应 prompt 模板
- 能在本地或团队工作流中复现一次简短的 AI 协作开发流程
