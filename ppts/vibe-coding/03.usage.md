# Cursor 核心技巧

## 1. 上下文管理 (@Symbols)
- **@Files**：引用特定文件
- **@Codebase**：全局向量检索（慎用，可能不准）
- **@Docs**：引用第三方文档

## 2. `.cursorrules` (项目的灵魂)
在项目根目录创建 `.cursorrules` 文件，定义 AI 的行为规范：

```markdown
# .cursorrules
- Always use TypeScript with strict mode.
- Use arrow functions for components.
- Style with TailwindCSS.
- For testing, use Jest + React Testing Library.
```

# 进阶能力：打破孤岛

## Agents & Sub Agents
- **主 Agent**：负责任务规划
- **子 Agent**：负责具体执行（如 @WebSearch, @Terminal）

## MCP (Model Context Protocol) 
让 AI 连接外部世界的标准协议：

- **读取数据库**：让 AI 直接查询 PostgreSQL Schema
- **访问 API**：让 AI 调用内部微服务
- **操作本地文件**：不仅是代码，甚至是 Excel/PDF

> MCP 让 AI 从 "Chatbot" 进化为真正的 "Assistant"。
