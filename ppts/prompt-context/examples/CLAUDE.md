# Demo Commands

Run individual demos from the `prompt-context` directory:

```bash
# Section 2.1: Zero-shot vs Few-shot
npm run demo:2.1

# Section 2.2: Chain of Thought / Extended Thinking
npm run demo:2.2

# Section 2.3: Context Window Strategies
npm run demo:2.3

# Section 2.4: Lost in the Middle
npm run demo:2.4

# Section 3.1: ICIO Framework
npm run demo:3.1

# Section 3.2: Claude Prompt Patterns
npm run demo:3.2

# Section 3.3: Tool Calling
npm run demo:3.3

# Section 4.1: Vue 3 Component Example
npm run demo:4.1

# Section 4.2: Bug Hunting - NN Client
npm run demo:4.2
```

Alternatively, run demos directly with node:

```bash
node examples/2.1.zero-vs-few-shot/main.js
node examples/2.2.cot-extended-thinking/main.js
# ... etc
```
