# Skill 工具生态

## @claudeai/skill 框架

官方的 Skill 开发框架，提供完整的开发工具：

```bash
npm install -g @claudeai/skill

skill init my-skill
skill dev
skill publish
```

# npm 生态

<v-clicks>

* **@claudeai/skill**: 官方框架

# Skill 发布与分发

<v-clicks>

1. **本地使用**: 放在项目或用户级配置目录
2. **npm 包发布**: 打包为 npm 包分发
3. **GitHub 分享**: 通过 GitHub 仓库分享
4. **Smithery.ai**: Skill 注册表

</v-clicks>
