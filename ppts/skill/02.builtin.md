# 内置技能

## Claude Code 内置 Skill

Claude Code 提供了多个开箱即用的内置 Skill，可以通过 `/` 命令调用：

---

# 常用内置技能

## /commit

自动创建 Git 提交

<v-clicks>

* 分析当前更改
* 生成提交信息
* 询问确认后执行

</v-clicks>

---

## /review-pr

代码审查

<v-clicks>

* 获取 PR 变更
* 进行代码审查
* 提供改进建议

</v-clicks>

---

## /test

生成测试

<v-clicks>

* 分析代码结构
* 生成单元测试
* 确保测试覆盖率

</v-clicks>

---

# 其他内置技能

<v-clicks>

* **/explain**: 解释代码功能
* **/refactor**: 重构代码
* **/fix**: 修复 Bug
* **/docs**: 生成文档

</v-clicks>

---

# 使用内置技能

直接在对话中输入 `/` 开头的命令即可：

```
/commit
/review-pr 123
/test generate tests for this function
```

<v-click>

查看所有内置技能：`/help`

</v-click>
