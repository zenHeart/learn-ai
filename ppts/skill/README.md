# SKILL

本系列旨在帮助开发者理解 Skill 作为可复用的工作流单元，掌握 Claude Code Skill 框架，并能够开发自定义 Skill。

## 核心内容

1. **SKILL 概述** - 什么是 Skill、与 MCP/Agent 的区别
2. **内置技能** - Claude Code 内置 Skills
3. **自定义 Skill** - YAML 配置开发
4. **技能生态** - GitHub 900+ Skills 分析
5. **实战案例** - 代码审查、自动化部署

## 预期收获

- 理解 Skill 作为可复用的工作流单元
- 掌握 Claude Code Skill 框架
- 能够开发自定义 Skill
- 能够使用 GitHub 主流 Skills

## 能力等级

本系列对应 **L5** 能力等级：能力扩展 - 能够通过 Skill 进行工作流编排
