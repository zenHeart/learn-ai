# 如何选择 Skills

<v-clicks>

1. **明确需求**: 需要解决什么问题？
2. **查看 Star 数**: 高 Star 通常更可靠
3. **检查维护**: 最近更新时间？
4. **阅读文档**: 是否清晰完整？
5. **测试验证**: 先在非生产环境测试

</v-clicks>
