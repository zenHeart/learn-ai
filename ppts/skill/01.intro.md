# 1. 什么是 Skill？

> **Skill** 是 Claude Code 中的可复用工作流单元，允许开发者将复杂的任务流程封装为可重复使用的自动化脚本。

::right::

## 对比 MCP/Agent

| 概念 | 定位 | 特点 |
|------|------|------|
| **MCP** | 能力扩展 | 提供工具/资源 |
| **Skill** | 工作流编排 | 封装任务流程 |
| **Agent** | 自主决策 | 复杂任务处理 |

---

# Skill 的核心价值

<v-clicks>

* **复用性**: 一次编写，多次使用
* **标准化**: 统一团队工作流
* **自动化**: 减少重复性操作
* **可分享**: 通过 npm 分发

</v-clicks>
