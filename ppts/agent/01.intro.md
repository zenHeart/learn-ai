# 1. 什么是 AI Agent？

> **AI Agent** 是能够自主规划、执行复杂任务的 AI 系统，具备感知、推理、决策和执行能力。

::right::

## Agent vs Copilot

| 特性 | Copilot | Agent |
|------|---------|-------|
| 交互模式 | 响应式 | 自主式 |
| 任务范围 | 单次请求 | 复杂任务 |
| 决策能力 | 用户主导 | 自主决策 |
| 典型场景 | 代码补全 | 端到端任务 |

---

# Agent 发展阶段

| 等级 | 描述 |
|------|------|
| L1 | 基础工具调用 - 单轮工具使用 |
| L2 | 链式调用 - 多步工具串联 |
| L3 | 自主规划 - 自动分解和执行任务 |
| L4 | 反思学习 - 从错误中学习和改进 |
| L5 | 多 Agent 协作 - 多个 Agent 协同工作 |
