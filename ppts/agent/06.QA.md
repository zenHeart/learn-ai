# 快速参考卡

## Agent Loop 核心

```
接收任务 → 规划分解 → 执行工具 → 评估结果 → 决策判断
                              ↓
                        成功 ← → 失败 → 反思调整
```

## Claude Code 模式

| 模式 | 说明 | 适用场景 |
|------|------|---------|
| loop | 自动执行 | 复杂任务 |
| stop | 暂停等待 | 关键步骤 |
| resume | 恢复执行 | 中断后继续 |
