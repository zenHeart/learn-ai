# AGENT

本系列旨在帮助开发者理解 AI Agent 的核心概念和架构，掌握 Agent Loop 原理，并能够构建多 Agent 协作系统。

## 核心内容

1. **AGENT 概述** - 什么是 AI Agent、Agent vs Copilot
2. **Agent Loop** - 规划-执行-评估-反思循环机制
3. **架构设计** - 规划器、记忆、工具、执行器组件
4. **Claude Agent** - Anthropic Agent SDK、Claude Code Agent 模式
5. **框架对比** - LangChain/AutoGen/CrewAI 实战
6. **实战案例** - 生产级 Agent 构建、多 Agent 协作

## 预期收获

- 深入理解 Agent Loop 原理
- 掌握 Claude Code Agent 模式
- 能够使用主流框架构建生产级 Agent
- 具备多 Agent 协作系统搭建能力

## 能力等级

本系列对应 **L6** 能力等级：任务编排 - 能够构建 Agent 处理复杂任务
