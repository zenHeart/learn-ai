# Prompt + Context Engineering

本系列旨在帮助开发者掌握提示词工程与上下文工程的核心概念与技巧，提升与 AI 模型的交互效率。

## 核心内容

1. **LLM 基础** - Token、Context Window、Transformer 原理
2. **Prompt 工程** - ICIO 框架、Zero-shot、Few-shot、CoT
3. **Context 工程** - 上下文管理、RAG、记忆机制
4. **Tool Calling** - 工具调用最佳实践
5. **实战案例** - 代码生成、知识问答等场景

## 预期收获

- 理解 LLM 核心原理
- 掌握 Prompt 工程核心框架和技巧
- 掌握上下文工程方法
- 能够构建高效的 AI 交互流程

## 能力等级

本系列对应 **L3** 能力等级：Prompt/Context 工程 - 能够优化提示词提升输出质量
