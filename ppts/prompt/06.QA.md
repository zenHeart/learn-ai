# 快速参考卡

## Prompt 核心技巧

| 技巧 | 场景 | 关键句式 |
|------|------|---------|
| Zero-shot | 简单任务 | 直接描述需求 |
| Few-shot | 需要格式 | 提供示例 |
| CoT | 推理任务 | "让我们一步步思考" |
| 角色设定 | 专业领域 | "你是一位..." |
| 格式控制 | 结构化输出 | "请用 JSON 格式" |

---

## ICIO 框架

```
Instruction (指令)     - 希望模型做什么
Context (上下文)      - 背景信息
Input Data (输入)     - 要处理的内容
Output Indicator (输出) - 输出格式
```

---

## 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 输出太短 | 提供示例/增加约束 |
| 格式错误 | 明确指定格式 |
| 幻觉严重 | 添加真实参考资料 |
| 角色偏离 | 加强角色设定 |
