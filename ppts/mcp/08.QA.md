# 快速参考卡

## MCP 核心概念

| 概念 | 说明 |
|------|------|
| Host | Claude Code 等客户端 |
| Client | MCP SDK 客户端 |
| Server | 工具提供者 |
| JSON-RPC | 通信协议 |

---

## 常用 MCP Server

| Server | 功能 |
|--------|------|
| @modelcontextprotocol/server-filesystem | 文件系统 |
| @modelcontextprotocol/server-github | GitHub API |
| @modelcontextprotocol/server-sqlite | SQLite 数据库 |
| @modelcontextprotocol/server-brave-search | Web 搜索 |

## 常见问题速查

| 问题 | 解决方案 |
|------|---------|
| 工具不可用 | 检查配置 + 重启 |
| 超时 | 增加超时/优化逻辑 |
| 权限错误 | 检查目录权限 |
