# 1. 什么是 MCP？

> **MCP (Model Context Protocol)** 是 AI 协作的"通用适配器"标准，让 AI 应用能够安全地与外部系统交互。

**官方定义**: MCP 是一个开放标准，用于将 AI 应用连接到外部系统。

::right::

## MCP 解决的问题

<v-clicks>

* **数据孤岛**: AI 无法直接访问你的数据
* **重复开发**: 每个 AI 工具都要单独集成
* **安全风险**: API 密钥暴露在多个地方
* **维护困难**: 更新一个 API 需要更新所有集成

</v-clicks>

---

# MCP vs 传统 API

| 特性 | 传统 API | MCP |
|------|---------|-----|
| 集成方式 | 每个客户端单独开发 | 标准化协议 |
| 工具发现 | 手动注册 | 自动发现 |
| 交互模式 | 请求-响应 | 双向对话 |
| 扩展性 | 低 | 高 |

<v-click>

**类比**: MCP 就像数字世界的 **USB-C 接口标准**

</v-click>
