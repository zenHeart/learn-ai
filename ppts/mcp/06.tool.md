# 工具集成

## Claude Code 中的 MCP

### MCP 工具前缀

Claude Code 中所有 MCP 工具都带有 `mcp__` 前缀：

```
mcp__context7__query-docs
mcp__github__list_issues
mcp__chrome-devtools__navigate_page
mcp__filesystem__read_file
```

---

# 配置 MCP Server

## 配置文件位置

* macOS: `~/Library/Application Support/Claude/claude_desktop_config.json`
* Windows: `%APPDATA%/Claude/claude_desktop_config.json`
* Linux: `~/.config/Claude/claude_desktop_config.json`

---

## 配置示例

<div class="grid grid-cols-3 gap-3 text-xs">

**1. 文件系统**

```json
{
  "mcpServers": {
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-filesystem", "/path"]
    }
  }
}
```

<v-click>

**2. GitHub**

```json
{
  "mcpServers": {
    "github": {
      "command": "npx",
      "args": ["-y", "@modelcontextprotocol/server-github"],
      "env": { "GITHUB_TOKEN": "xxx" }
    }
  }
}
```

</v-click>

<v-click>

**3. 自定义 Server**

```json
{
  "mcpServers": {
    "my-server": {
      "command": "node",
      "args": ["/path/to/server.js"],
      "env": { "API_KEY": "xxx" }
    }
  }
}
```

</v-click>

</div>

---

## 配置步骤详解

<v-clicks>

1. **打开配置文件**: 使用编辑器打开 `claude_desktop_config.json`

2. **添加 MCP 配置**: 在 `mcpServers` 对象中添加新 Server

3. **指定命令**: 使用 `command` 和 `args` 定义启动命令

4. **设置环境变量**: 在 `env` 中配置 API 密钥等敏感信息

5. **重启 Claude Code**: 配置更改后需要重启生效

6. **验证配置**: 运行 `claude mcp list` 确认工具已注册

</v-clicks>

# 常用 MCP 工具生态

<v-clicks>

* **context7**: 文档查询
* **github**: GitHub API 集成
* **chrome-devtools**: 浏览器自动化
* **filesystem**: 文件系统操作
* **postgres**: 数据库查询
* **google-maps**: 地图服务

</v-clicks>

---

# 工具发现机制

当 Claude Code 连接到 MCP Server 时：

1. Server 发送工具列表
2. Claude Code 自动注册所有工具
3. AI 自动识别可用工具
4. 根据需求自动调用工具

<v-click>

这就是 Claude Code 中看到 `mcp__` 前缀工具的原因！

</v-click>
