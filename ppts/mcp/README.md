# MCP (Model Context Protocol)

本系列旨在帮助开发者理解 MCP 协议架构，掌握 MCP Server 开发，并能在 Claude Code 中有效使用 MCP 工具。

## 核心内容

1. **MCP 概述** - 什么是 MCP、解决的问题
2. **架构设计** - Host/Client/Server 三方架构
3. **协议详解** - JSON-RPC 2.0、工具/资源/提示词
4. **Server 开发 TypeScript** - SDK 实战
5. **Server 开发 Python** - FastMCP 最佳实践
6. **工具集成** - Claude Code MCP 配置
7. **实战案例** - 主流 MCP Server 分析

## 预期收获

- 深入理解 MCP 协议架构和通信机制
- 掌握 TypeScript SDK 开发 MCP Server
- 掌握 Python SDK (FastMCP) 开发
- 能够开发生产级 MCP Server
- 掌握 Claude Code MCP 配置

## 能力等级

本系列对应 **L5** 能力等级：能力扩展 - 能够通过 MCP 扩展 AI 能力
