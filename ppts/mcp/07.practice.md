# 故障排除

## 常见问题 1：MCP Server 无法启动

**症状**: 配置 MCP 后工具不可用

**排查步骤**:
- 检查配置文件语法
- 查看 MCP 日志
- 手动测试 Server 启动

**常见原因**:
- Node.js 版本不兼容
- 依赖未正确安装
- 端口被占用

---

## 常见问题 2：工具调用超时

**症状**: MCP 工具调用长时间无响应

**排查步骤**:
- 检查网络连接
- 查看 Server 日志

**解决方案**:
- 增加超时配置
- 检查 Server 资源占用
- 优化工具执行逻辑

---

## 常见问题 3：权限错误

**症状**: 提示权限不足

**解决方案**:
- 检查 MCP 配置中的权限设置
- 确认工作目录访问权限
- 查看 Claude Code 日志获取详情
